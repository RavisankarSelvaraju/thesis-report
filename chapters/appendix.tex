    \subsection{Velocity}
        When the measurement function is differentiated with respect to the body velocity,
        
    $$\frac{\partial h}{\partial v_{body}^{body}} = - \frac{\partial v_{body}^{body}}{\partial v_{body}^{body}} - \frac{\partial (\omega_{body}^{body} \times r_{pc}^{body})}{\partial v_{body}^{body}} = -I - 0 = -I_{3\times3}$$
